# Webpack 4 Boilerplate using SASS

- Webpack v4
- SASS/CSS
- Webpack-dev-server
- Minimizer for Javascript & CSS
- Img loader
- Cross-env (to set the variable NODE_ENV)

## Getting Started

```
$ npm install
```

### Prerequisites

You just need to install node.js

### Running dev environment

```
$ npm start
```

### Running prod environment

```
$ npm run build
```
